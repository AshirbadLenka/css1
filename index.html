What is CSS?

  1. CSS stands for Cascading style sheets.

  2. CSS describes how HTML elements are to be displayed

  3. Css is not necessary CSS saves lot of work. It can control the layout of multiple web pages all at once.

  4. External stylessheets are stored in CSS files.

CSS can be added to HTML document in 3 ways:

  • Inline - by using the style attribute inside the HTML documents.
  • Internal - by using a style element in the head section.
  • External - by using a link element to an external CSS file.
  • the most common way to add CSS, is to keep the styles in external CSS files. However , in this tutorial we will use inline and internal styles, because it is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID,the difference is that CSS class applies a style to multiple elements.ID , on the other hand,applies a styles a style to one unique element.ID is also special inthat you can use a special URL to link directly to an element and it's used by javascript.

In CSS, selectors are used to target a specific element or range of elements on aweb page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
